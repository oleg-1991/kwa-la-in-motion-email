# KWA – LA In Motion (Email template)

HTML email newsletter for Kimberlina Whettam & Associates.

## Features

- 600px main container, table-based layout
- Responsive tweaks for mobile (max-width: 599px)
- Safe for major email clients (Gmail, Outlook, Apple Mail, etc.)
- Images stored separately in `/images` for easier management
- Ready to paste into Constant Contact as custom HTML

## Files

- `index.html` – main email template
- `images/` – content images used in the layout
